COSTECH IT Support
Bienvenue sur le dépôt officiel de COSTECH IT Support, une plateforme web conçue pour faciliter la mise en relation entre les entreprises (ou particuliers) et mes services d'expert en solutions informatiques basés à Douala, Cameroun.

🚀 À propos du projet
COSTECH IT Support est une application web dynamique et responsive, développée pour simplifier la demande d'assistance technique. Elle permet aux clients de soumettre leurs problèmes via un formulaire intégré qui envoie instantanément les détails vers WhatsApp, garantissant ainsi une réactivité maximale.

🛠 Services proposés
Je propose des solutions informatiques complètes, notamment :

Développement Web : Création d'applications sur mesure, sites web vitrines et plateformes métiers.

Maintenance Informatique : Dépannage matériel (hardware) et optimisation logicielle pour les parcs informatiques d'entreprise.

Réseaux & Télécoms : Installation, configuration et sécurisation de réseaux locaux.

Récupération de données : Assistance pour la sécurisation et la restauration de vos fichiers critiques.

💻 Stack Technique
Le projet a été développé avec les technologies suivantes :

Frontend : HTML5, CSS3 (avec approche responsive), JavaScript (ES6+).

Intégration : FontAwesome pour les icônes.

Communication : Intégration directe avec l'API WhatsApp pour une gestion fluide des requêtes clients.
